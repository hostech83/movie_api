Movie API Documentation

Endpoints

GET /movies

Fetches all movies in the database.

GET /movies

GET /movies/:title

Fetches a single movie by title.

GET /movies/Inception

GET /genres/:name

Fetches movies by genre name.

GET /genres/Sci-Fi

GET /genres/:name

Fetches movies by genre name.

GET /genres/Drama

GET /directors/:name

Fetches movies by director name.

GET /directors/Christopher%20Nolan

GET /users

Fetches all users.

GET /users

POST /users

Creates a new user.

POST /users

Example request body:

{
  "username": "num_paki",
  "password": "test123",
  "email": "num_paki@example.com"
}

PUT /users/:username

Updates user information by username.

PUT /users/new_user

Example request body:

{
  "password": "new_passwordtest123"
  "email": "test_user@example.com"
}

POST /users/:username/movies/:movieId

Adds a movie to a user's favorites.

POST /users/new_user/movies/667a55057c2505b791cdcdf6

DELETE /users/:username:emiley_green

Deletes a user by username.

DELETE /users/new_user
